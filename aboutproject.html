    • Reflections:

    • This project made me develop my skills not only in HTML and CSS, but also in time management. In fact, time was probably my biggest adversary in this project. However, I managed to push through on a project where I'm not going to lie, I would have probably procrasrinated to. My main solution for this was learning how I worked. After I did, I found out how to be the most efficient I could be and thus, finish this long project in the time given. Also, I decided to make tables for the players, which we did not learn in class. A great site, called w3schools.com really helped me, as well as confirm any suspicions I had otherwise. Other classmates were also crucial, especially for the design. They would give me some constructive critisism to help me develop the backdrop as well as the grid of this site.
      One thing I know about myself, is that I will always improvise, even if there is a plan. I had a plan, and I made some pretty big changes from it. for example, this document was supposed to be what my main page was to look like. Although there are some similarities, there are many differences too. However, that's probably a good thing. Also, the Dortmund team colors are black and yellow, so it was quite obvious that the color scheme had to match that too. In fact, the backdrop for the grid is from the official Dortmund website. Like I said, I learned a lot about my own time management style. I found a study that shows that working (or studying in that example) for hours on end is a bad idea. I used that knowledge to help me work on this project.
      I like my website because the amount of work that I put into this thing really shows the passion that I have for the sport of soccer as well as my own team, Borussia Dortmund. I think that other people would also like this site because if they see how much work I put into something that I like, maybe it will motivate them to do the same with something that they all like. In fact, one of my biggest regrets about this site is that I never learned JavaScript early enough to implement it into this site, as I wanted to add more detail. I think that this site will do quite well, because of the amount of CSS and HTML that I used. However, I find that most of my attention was drawn to the tables of the players. If it wasn't, who knows how well I could've done.
